# 月舟 tsuki-bune

月舟(tsuki-bune)は、**日記をHTMLで書くための軽いWebアプリ**です。

WordやGoogle Docsのように見たまま書けて、MarkdownのようにAIやGitにも扱いやすいファイルを残すことを目指しています。

アカウントは不要です。日記はユーザー自身のローカルフォルダに保存します。月舟は日記内容を勝手に外部送信しません。

## なぜ作るのか

AIはとても賢くなっていますが、その人の背景、日々の生活、考え方、好み、過去の出来事を知らないと、気の利いた返答ができないことがあります。

自分の情報、日記、生活ログ、考えたことを「人生レポジトリ」として残しておけば、将来AIに自分の背景を伝えやすくなります。

しかし、今の人生レポジトリ運用はエンジニアに寄りがちです。GitHub、VS Code、Markdownに慣れていない人にとっては、始めるハードルが高いです。

月舟は、そのハードルを下げるためのアプリです。文章を書く体験はなるべくWordのように直感的にしつつ、保存形式はAIやGitにも読みやすいHTMLにします。

## 月舟が大事にすること

- サクサク軽く動くこと
- 開いてすぐ書けること
- アカウントなしで使えること
- 日記内容を勝手に外部送信しないこと
- HTMLとして読める形で保存すること
- パソコン初心者にも迷いにくいこと
- 機能を増やしすぎず、明瞭であること
- 本文を書く体験を邪魔しないこと

## 主な機能

- 新規日記作成
- WYSIWYG編集
- 太字、下線、取り消し線
- 文字色
- 蛍光ペン
- 文字サイズ
- 見出し
- 箇条書き
- 画像貼り付け
- HTML保存
- `Ctrl+S` / `Cmd+S` 保存
- 日記フォルダ選択
- フォルダ内HTML一覧
- ファイル名検索
- 貼り付け画像の `assets/` 保存

## 基本の使い方

1. 月舟を開きます。
2. `日記フォルダ` を押して、日記を保存するフォルダを選びます。
3. `新規日記` で書き始めます。
4. 必要に応じて、太字、下線、文字色、蛍光ペン、画像貼り付けなどを使います。
5. `HTML保存` または `Ctrl+S` / `Cmd+S` で保存します。

保存したHTMLは、ブラウザやエディタでも開けます。月舟で再度読み込んで編集することもできます。

## ブラウザ対応

Chrome / Edgeなど、File System Access APIに対応したブラウザでは、日記フォルダへ直接保存できます。

対応していないブラウザでも、HTML読込とダウンロード保存は利用できます。

## データとプライバシー

月舟は、ユーザーの文章と画像をユーザー自身のものとして扱います。

- 日記内容を月舟側が勝手に外部送信しません。
- 日記内容を運営側サーバーへ保存しません。
- アカウントなしで使えることを重視します。
- 保存したHTMLと画像は、ユーザーのローカルフォルダに残ります。
- 月舟を使わなくなっても、HTMLと `assets/` の画像は残ります。

ブラウザ内の一時保存は、編集中内容を守るための保険として使います。ただし、正式な保存はHTMLファイルとして保存する操作です。

## 保存形式

月舟は、日記をHTMLとして保存します。

画像はHTMLに巨大なData URLとして埋め込まず、日記フォルダ内の `assets/` に保存します。HTMLからは `./assets/...` の相対パスで参照します。

これにより、HTML本文が重くなりにくく、Gitの差分やAIへの読み込みでも扱いやすくなります。

## 技術スタック

月舟は、まず小さく軽く動かすために、Vanilla HTML/CSS/JavaScriptで実装しています。

- フロントエンド: Vanilla HTML/CSS/JavaScript
- 編集: `contenteditable` + ブラウザ標準編集機能
- 保存: File System Access API / HTMLダウンロード
- 画像: `assets/` への外部保存
- ローカル開発: Docker + Nginx

今後、編集体験を本格化する段階では、TipTap/ProseMirrorなどへの移行も検討します。

## ローカル実行

Dockerを使う場合:

```bash
docker compose up --build
```

ブラウザで開きます。

```text
http://localhost:8080
```

Dockerを使わない場合:

```bash
python3 -m http.server 5173
```

ブラウザで開きます。

```text
http://localhost:5173
```
