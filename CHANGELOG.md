# Changelog

## [0.1.0] - 2026-05-12

### Added

- WYSIWYG diary editor.
- HTML save and download fallback.
- `Ctrl+S` / `Cmd+S` save shortcut.
- Local diary folder support with File System Access API.
- Folder HTML list with filename search.
- Basic formatting toolbar.
- Text color and highlight palettes.
- Font size and heading menus.
- Image paste support with `assets/` folder storage.
- Unused pasted image cleanup on save.
- About dialog from the brand icon.
- Public-release README.
